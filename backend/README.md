## Development

Start the project:

```bash
bun dev
```

## Migrations

Push schema to Database:

```bash
bunx drizzle-kit push
```
Generate new migration:

```bash
bunx drizzle-kit generate
```
Apply migrations:

```bash
bunx drizzle-kit migrate
```

## Production

Run project in `production` mode:

```bash
bun start
```
